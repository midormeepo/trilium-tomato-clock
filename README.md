## 安装和更新
1. 从发行版中下载[Tomato.zip](https://github.com/midormeepo/trilium-tomato-clock/releases)压缩包；
2. 导入到笔记，选择下载好的压缩包并取消勾选“安全导入”；
3. 在需要的笔记中添加 ```#tomato``` 属性 (需要切换或刷新笔记显示番茄钟)。
4. *更新直接按上述步骤覆盖即可。
